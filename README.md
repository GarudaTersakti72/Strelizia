# Install Strelizia

- apt-get update

- apt-get install git

- git clone https://github.com/GarudaTersakti72/Strelizia/

- cd Strelizia

- chmod +x liz-install.sh

- ./liz-install.sh

#### Almost Done! type ./liz.sh for Run Strelizia.
